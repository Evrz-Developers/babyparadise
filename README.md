# babyparadise

An e-commerce app built with Flutter.

## Features

- [x] Login with Google
- [x] Register with email and password
- [x] Forgot password
- [x] Reset password
- [x] Admin panel
- [x] User panel
- [ ] Product management:
  - [x] Add product
  - [ ] Edit product
  - [ ] Delete product
- [ ] Cart management
- [ ] Order tracking
